## 2. 技术指标分析（MACD + KDJ，量比辅助）

### 量比算法
- **公式**：量比 = 当前累计成交量 ÷ 最近20日平均全天成交量
- **盘中预估**：如果是盘中分析，因为当天交易还没有结束，量比偏低，参考价值低。只有在执行 `/daily_archive` 命令时，才完全采信量比指标

### 实战信号统计（数据驱动）

基于最近30个交易日的数据统计，有以下发现：

| 信号 | 次日平均涨跌 | 上涨概率 | 实战结论 |
|------|-------------|---------|---------|
| **J < 0（严重超卖）** | **+0.57%** | **56.6%** | 所有单一信号中效果最佳，超跌反弹的核心标志 |
| KDJ低位金叉（K<30,D<30） | +0.50% | 47.6% | 信号有效，但需配合板块和量能 |
| MACD金叉 | +0.21% | 46.9% | **金叉当日不能追涨**，短期动能已部分释放 |
| J > 100 | +0.18% | 47.8% | 高位追涨次日无明显优势，需警惕回落 |
| 量比 > 1.5 | +0.06% | 42.3% | 放量不等于次日会继续涨，只作为资金确认 |

**核心结论**：
- **J < 0 是短线最好的单一信号**，当个股属于近期热点但短期被错杀时，反弹概率最高
- **MACD金叉次日胜率低于50%**，说明它是趋势确认指标，而非追涨信号。真正的高胜率买点在**金叉前一日**或**金叉后的首次回踩**
- **DIF > 0 是持续性的分水岭**：DIF为正的强势股，回调即是买点；DIF为负的股票，大涨多为超跌反抽，不可追高

### MACD 规则（定方向）

| 信号 | 条件 | 操作 |
|-----|------|------|
| 多头启动 | DIF 上穿 DEA 且 DIF 从负转正（零轴下方金叉） | 建仓信号 |
| 趋势加速 | DIF > 0 且 DIF 上穿 DEA（零轴上方金叉） | 加仓信号 |
| 趋势结束 | DIF 下穿 DEA 且 DIF 从正转负 | 清仓信号 |
| 顶背离 | 股价创新高，DIF 未创新高 | 减仓/止盈 |
| 底背离 | 股价创新低，DIF 未创新低 | 关注买入 |

**MACD 柱状图**：红柱放大=动能增强，红柱缩短=动能减弱，绿柱反之。

### MACD 买点时机修正（关键）

**最好的买点在"发榜前一天"，不是"发榜当天"**：

- **高胜率买点**：DIF 接近 DEA（两者差值 < 0.2）+ KDJ J < 20 + 量比 > 1.0
  - 典型案例：数据港 4月7日 DIF=-0.197（距金叉仅一步之遥）+ J=8.5 → 次日涨停
  - 典型案例：中信证券 4月10日 DIF=-0.49 尚未金叉，但已放量大涨 7.25%
- **次高胜率买点**：DIF > 0 的强势股，首次回踩 MA5/MA10 或 J 从高位回落至 50-70 区间
  - 典型案例：新易盛 4月9日回踩至 490元（J 从 89 回落至 67），次日大涨 6.63%
- **低胜率买点**：MACD 金叉当天追涨。统计表明金叉次日胜率仅 46.9%，因为短期动能已被提前消化

**执行原则**：看到 MACD 即将金叉时，如果 KDJ 尚未进入极低区域，宁可错过也不要追高；等待回踩确认再入场。

### KDJ 规则（找买卖点）

| 信号 | 条件 | 操作 |
|-----|------|------|
| 低位金叉 | K < 30, D < 30, K 上穿 D | 买入 |
| 高位死叉 | K > 80, D > 80, K 下穿 D | 卖出 |
| 严重超买 | J > 100 连续 2 天 | 减仓 |
| 严重超卖 | J < 0 | 可考虑买入，做超短线 |

### 共振交易规则（高胜率）

**买入条件分级（按优先级）**：

**A级买点：DIF > 0 强势股的回踩**（适用于趋势龙头）
- MACD：DIF > 0 且均线多头排列（MA5 > MA10 > MA20 或 MA5 回踩 MA10/MA20）
- KDJ：J 从高位（>80）回落至 50-70 区间，或 K、D 尚未死叉
- 价格：收盘价 > MA20
- 量能：量比 > 0.8，回踩时缩量更好
- **止损**：MA10 或 MA20 破位，或固定止损 **-5% ~ -8%**。对 DIF > 15 的高波动龙头，**严禁使用 -3% 窄止损**（历史教训：新易盛曾因 -3.2% 止损被洗盘洗出，错失后续 14% 涨幅）

**B级买点：DIF 即将金叉 + KDJ 超跌共振**（适用于短期被错杀的个股）
- MACD：DIF 接近 DEA（差值 < 0.2），即将金叉或刚金叉，最好能见到 DIF 由负转正的预期
- KDJ：J < 20（越低越好，J < 0 最佳）
- 价格：不要求必须 > MA20，但不能偏离 MA20 过远（< -8% 需警惕趋势破坏）
- 量能：量比 > 1.0，放量确认资金回流
- **止损**：买入当日最低点或 -5% 严格止损

**C级买点：J < 0 的极端超卖反弹**
- KDJ：J < 0
- 板块：个股不是边缘弱势股
- 量能：下跌过程缩量，说明抛压枯竭
- **预期**：短线套利，不恋战，有盈利即走；次日不继续走强则离场

**卖出条件（满足任一）**：
1. MACD 顶背离出现
2. KDJ 高位死叉（K > 70, D > 70）
3. 亏损超过 -8% 强制止损
4. 盈利超过 +20% 且 KDJ J > 100（动态止盈）

### 注意事项

- **趋势优先**：MACD 定方向，KDJ 找时机；零轴下方以轻仓或空仓为主
- **过滤震荡**：KDJ 在 30-70 区间频繁交叉时，不参与交易
- **背离确认**：顶/底背离需配合成交量变化，单一背离不操作
- **单边行情**：KDJ 钝化时（长期超买/超卖），以 MACD 和均线为准

## 3. 大盘研判框架

**核心定位：大盘只回答"能买多少"，不回答"能不能买"。个股信号触发立即执行，不因大盘弱势犹豫。**

| 状态 | 判断依据 | 仓位上限 | 单票上限 |
|------|---------|---------|---------|
| **强势** | DIF > 0 且 收盘价 > MA20 | 80% | 30% |
| **震荡** | DIF 在 -0.5 ~ +0.5 之间 | 65% | 30% |
| **弱势** | DIF < -0.5 且 收盘价 < MA20 | 50% | 20% |

**执行规则：** 个股信号触发 → 立即执行；总仓位超限则按上限买入，不猜测底部。
