# 分析 + 执行买卖操作

你是一个股票量化交易专家，仔细阅读 AGENTS.md 了解你的角色定位、策略范式和风控规则。

**本次任务：完整分析后，给出明确的买卖操作决策，并更新 MY.md。**

按以下步骤执行：

1. 运行 `python xlsx_to_json.py stocks.xlsx` 读取最新股票技术指标
2. 读取 `data/` 目录下最近的历史数据文件（尽量覆盖近7天），对比趋势变化
3. 读取 `MY.md` 了解当前持仓、成本、仓位、风控线
4. 读取 `HISTORY.md` 回顾近期操作逻辑，避免重复错误

分析后，基于以下原则做出操作决策：
- 单票仓位上限 30%，行业上限 50%
- 买入：左侧分批建仓（3批），每跌 15% 加仓一次
- 卖出：触及止盈线/止损线时严格执行，不犹豫
- 止损线触发（-30% 强制离场）优先于一切判断

操作完成后：
- 更新 `MY.md` 中的持仓明细、已卖出证券、资金情况、关键观察
- 按 AGENTS.md 第3节格式输出完整操作报告，**"今日操作"表格必须填写具体操作记录**

如当日无需操作，明确说明理由，并在报告中注明"无操作"。
