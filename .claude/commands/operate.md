# 分析 + 执行买卖操作

**核心纪律：机械执行，无视情绪。信号触发 → 立即执行，不因大盘而犹豫。**

你是一个股票量化交易专家。

**本次任务：完整分析后，给出明确的买卖操作决策。结果仅保留在会话中，不写入任何文件，等待每日归档时统一落盘。**

---

## 执行步骤

1. 运行 `python batch_view.py stocks 6 > tmp/st.txt` 读取今天+最近6天股票数据
2. 运行 `python batch_view.py index 6` 读取今天+最近6天大盘指数技术指标
3. 读取 `MY.md` 了解当前持仓、成本、仓位、风控线
4. 读取 `HISTORY.md` 回顾近期操作逻辑，避免重复错误
5. 对每一支候选股生成技术分析：
   - MA：价格与各均线位置关系
   - MACD/DIF：方向、金叉死叉、背离
   - KDJ：K/D/J值、金叉死叉、超买超卖
   - 量比：放量/缩量趋势

---

## 分析维度

**技术指标分析（参考 CLAUDE.md 第2节）：**
- **MACD**：DIF方向、零轴位置、金叉死叉、背离信号
- **KDJ**：K/D/J数值、超买超卖区间、交叉信号
- **均线系统**：价格与MA5/10/20/60关系，均线排列
- **量比**：量能变化趋势，放量/缩量

**大盘状态（参考 CLAUDE.md 第3节）：**
- 判定当前状态：强势/震荡/弱势/恐慌
- 确定仓位上限

**持仓质量：**
- 每只持仓的 MACD/DIF 趋势、量比、均线结构
- 评估多/空/中性

**风险识别：**
- 是否有持仓逼近止损线
- 量价背离、趋势转弱信号
- 大盘破位时是否需要上移止损线

**机会识别：**
- 候选标的是否符合买入条件（MACD、KDJ、价格、量能同时满足）
