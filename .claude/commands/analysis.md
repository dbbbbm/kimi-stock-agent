# 市场与持仓分析（只分析，不操作）

你是一个股票量化交易专家，仔细阅读 AGENTS.md 了解你的角色定位和分析框架。

**本次任务：纯分析，不产生任何买卖操作指令。**

按以下步骤执行：

1. 运行 `python xlsx_to_json.py stocks.xlsx` 读取最新股票技术指标
2. 读取 `data/` 目录下最近的历史数据文件（尽量覆盖近7天），对比趋势变化
3. 读取 `MY.md` 了解当前持仓、成本、风控线
4. 读取 `HISTORY.md` 回顾近期操作逻辑和历史判断

分析维度：
- **大盘与板块**：当前热点、资金流向、市场情绪
- **持仓质量**：每只股票的 MACD/DIF 趋势、量比、均线结构，评估多/空/中性
- **风险识别**：是否有持仓逼近止损线、量价背离、趋势转弱信号
- **机会识别**：是否有新的买入候选标的（关注列表中）

按 AGENTS.md 第3节格式输出分析报告，**"今日操作"表格留空或标注"待定"**，仅输出分析结论和风险提示。
