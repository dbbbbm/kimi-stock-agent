# 市场与持仓分析（只分析，不操作）

你是一个股票量化交易专家，仔细阅读 `CLAUDE.md` 了解你的角色定位和分析框架。

**本次任务：纯分析，不产生任何买卖操作指令。**

按以下步骤执行：

1. 运行 `python xlsx_to_json.py stocks.xlsx` 读取最新股票技术指标
2. 运行 `python xlsx_to_json.py index.xlsx` 读取今日大盘指数技术指标
3. 读取 `indices/` 目录下最近 7 天的历史大盘快照（每天取最新一个文件），对比大盘 DIF 趋势、量比变化、MA20 站位
4. 读取 `data/` 目录下最近的历史股票数据文件（尽量覆盖近7天），对比个股趋势变化
5. 读取 `MY.md` 了解当前持仓、成本、风控线
6. 读取 `HISTORY.md` 回顾近期操作逻辑和历史判断

分析维度：
- **大盘状态**：依据 `CLAUDE.md` 第4节框架，判定当前大盘处于强势/震荡/弱势/恐慌哪种状态，以及对应的仓位上限
- **指数历史趋势**：对比近7天上证/沪深300 DIF 变化方向（加速上行/减速/横盘/加速下行），量比是否持续萎缩，价格是否在 MA20 上下方；创业板独立行情是否成立
- **持仓质量**：每只股票的 MACD/DIF 趋势、量比、均线结构，评估多/空/中性
- **风险识别**：是否有持仓逼近止损线、量价背离、趋势转弱信号；大盘破位时是否需要上移止损线
- **机会识别**：是否有新的买入候选标的（关注列表中），当前大盘状态是否允许建仓

按 `CLAUDE.md` 第3节格式输出分析报告，**"今日操作"表格留空或标注"待定"**，仅输出分析结论和风险提示。

报告中"市场分析"部分须包含：
- 大盘状态评级（强势/震荡/弱势/恐慌）及判断依据
- 当前允许的最大仓位上限
