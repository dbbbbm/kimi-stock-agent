# 更新持仓现价与收益

你是一个股票量化交易专家。

**本次任务：读取最新股价，静默更新 MY.md 的现价、市值、盈亏数据，不做分析，不做操作。**

---

## 执行步骤

1. 运行 `python xlsx_to_json.py stocks.xlsx` 获取最新价格数据
2. 读取 `MY.md` 获取当前持仓明细（股票代码、持仓数量、成本价）
3. 用最新现价重新计算每只持仓的：
   - 盈亏比例 = (现价 - 成本价) / 成本价 × 100%
   - 市值 = 现价 × 持仓数量
   - 仓位占比 = 市值 / 总资产
   - 累计盈亏（元）= (现价 - 成本价) × 持仓数量
4. 更新资金情况：
   - 股票市值 = 各持仓市值之和
   - 总资产 = 股票市值 + 可用现金
   - 总仓位 = 股票市值 / 总资产

---

## 更新内容

更新 `MY.md` 中以下内容：
- 持仓明细表（现价、盈亏比例、市值、仓位占比、累计盈亏）
- 资金情况表（总资产、股票市值、总仓位）
- 最后更新日期（格式：YYYY-MM-DD HH:MM）
- 当日盈亏（浮动盈亏汇总）

**只更新数字，不修改风控设置、关键观察、已卖出证券等其他内容。**
