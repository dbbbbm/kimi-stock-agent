# 5月8日A股盘中总结

**截至午间休市（11:30）**，今日A股三大指数集体回调，呈现"指数走弱、题材轮动"格局。沪指跌0.43%报4162点，深成指跌约1%，创业板指跌1.01%，科创50跌幅扩大至超3%。两市半日成交额约1.5万亿元，较昨日同期缩量超300亿元。

---

### 1. 领涨板块

今日盘面热点快速轮动，科技成长内部分化加剧，机器人与AI应用端逆势活跃，而前期强势的半导体芯片大幅回调：

| 板块/概念 | 表现 | 驱动逻辑 |
|---|---|---|
| **人形机器人** | 绿的谐波涨超16%，三花智控涨超5%，机器人ETF涨超3% | 特斯拉、宇树等企业量产时间表明确，Q2正式订单下达，产业链进入小批量交付 |
| **汽车零部件** | 凌云股份、祥鑫科技、五洲新春、飞龙股份涨停，零部件ETF涨1.88% | 连续4日资金净流入，智能驾驶+轻量化中期逻辑清晰 |
| **商业航天** | 电科蓝天20cm涨停刷新上市高点，大业股份3连板，航天长峰等涨停 | 未来3-5年商业航天高密度发射期，发射场年能力有望突破60次 |
| **AI应用/软件** | 中国软件直线涨停，引力传媒2连板，南威软件2天2板 | 海外AI软件业绩超预期，腾讯云明日上调AI算力刊例价5% |
| **房地产** | 宁波富达、京投发展等多股涨停 | 政策预期与板块轮动补涨 |
| **游戏** | 完美世界涨停，中青宝涨超10% | 新品周期与AI应用落地预期 |

**跌幅居前**：半导体芯片（寒武纪跌超6%，海光信息、兆易创新跌超4%）、煤炭、石油石化、锂电电解液。

### 2. 主力资金流向

**早盘主力资金呈现"逃离高位科技、切入低位制造"的调仓特征：**

**净流入方向：**
- **行业板块**：交运设备、通用设备、IT服务获主力净流入
- **个股**：浪潮信息（+11.38亿）、胜宏科技、云南锗业、立讯精密净流入居前
- **概念**：机器人、汽车零部件、软件服务获资金青睐

**净流出方向：**
- **行业板块**：电子板块净流出超138亿元，电新行业、半导体遭大幅抛售
- **个股**：兆易创新（-14亿）、海光信息、中国长城、东山精密净流出额居前
- **宽基指数**：沪市主力整体净流出265.7亿元，超大单与大单同步流出

**资金情绪**：5月以来市场资金情绪明显回暖，但今日早盘出现获利回吐，科技主题ETF资金向中小盘宽基及机器人、AI应用等低位方向迁移。

### 3. 重要新闻

**宏观与政策：**
- **中美高层沟通**：外交部确认双方就特朗普总统访华保持沟通，华尔街日报称下周会晤期间可能启动AI官方讨论
- **欧盟对华限制**：欧盟首次将中国划定为"高风险国家"，拟禁止对使用中国逆变器的项目提供资金支持，商务部坚决反对
- **巴西免签**：自5月11日起对持普通护照中国公民实施免签入境，每次最长停留30天
- **央行增持黄金**：4月末黄金储备7,464万盎司，环比增加26万盎司，连续18个月扩大

**产业与公司：**
- **中国移动移动云大会**：在苏州召开，宣布开放万亿级Token服务体验包，升级模型服务平台MoMA，布局吉瓦级AI数据园区
- **国产大模型融资**：阶跃星辰将完成近25亿美元融资，拆除红筹架构加速赴港IPO；月之暗面（Kimi）完成约20亿美元融资，投后估值超200亿美元
- **算力涨价潮**：腾讯云宣布5月9日起对AI算力、容器服务等相关产品刊例价统一上调5%，国内三大CSP集体终结"以价换量"策略
- **存储器超级周期**：德明利、佰维存储、江波龙、澜起科技等多家存储产业链公司一季度净利润创单季历史新高
- **退市加速**：年内已有22家公司锁定退市，其中13家触及财务类退市指标，资本市场"有进有出"生态加速形成

**外围市场：**
- **美股收跌**：隔夜道指跌0.63%，纳指跌0.13%，纳斯达克中国金龙指数收跌1.33%
- **地缘冲突**：美伊在霍尔木兹海峡交火，原油运价指数持续运行在历史高位，WTI原油突破98美元/桶
