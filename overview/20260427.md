# 2026年4月27日 A股大盘总结

---

## 一、指数表现

今日A股三大指数走势分化，科创综指表现强劲，大涨逾2%，逼近历史最高点 。上证指数围绕4070-4090点区间震荡，创业板指盘中一度跌超1.5% 。两市成交较上周五缩量10%-15%，降至1万亿元以下，资金观望情绪上升 。

## 三、主力资金流向

- **半导体方向**：半导体设备ETF（159558）盘中换手7.24%，成交4.30亿元，规模和份额均创成立以来新高 ；科创芯片设计ETF、科创芯片ETF盘中涨超4% 。
- **北向资金**：受外围市场波动与高位股调整影响，今日小幅净流出，转向电力、银行等高股息蓝筹避险 。
- **锂电池/固态电池概念**：锂电池概念获主力资金净流入126.08亿，固态电池净流入110.85亿，钠离子电池净流入66.69亿（注：该数据为4月24日收盘数据，趋势延续） 。
- **风格切换**：资金从高位CPO、AI算力等题材股流出，转向业绩确定、估值低位的硬科技与新质生产力主线 。

## 四、重要新闻

1. **央行MLF净回笼2000亿元**：4月27日，央行开展4000亿元MLF操作，本月到期6000亿元，实现净回笼2000亿元，结束此前连续13个月加量续做；本月中期流动性净回笼达6000亿元 。

2. **4家公司拉响退市警报**：4月26日晚间，4家A股上市公司发布将被或可能被实施退市风险警示的公告，其中3家因营收及利润数据触及相关情形 。

3. **东方甄选核心主播集体离职**：4名主播明明、天权、中灿、林林陆续宣布离开东方甄选，俞敏洪公开致歉 。

4. **国际油价显著上涨**：布伦特原油突破101美元/桶，WTI原油突破96美元/桶，高盛将2026年Q4布伦特原油价格预测上调至90美元/桶 。

5. **证监会部署2026年打击财务造假专项行动**：4月24日部署，提出优化上市公司监管执法信息统筹发布，提升执法威慑效果 。

6. **OpenAI计划自研手机**：天风国际分析师郭明錤发文称，OpenAI正与联发科、高通合作开发手机处理器，立讯精密为独家制造商，预计2028年量产 。

---

## 五、市场特征总结

今日市场核心特征是**"风格切换，业绩为王"**：前期领涨的高位CPO、AI算力等题材股因"高估值、无业绩、获利盘多"持续退潮，资金转向业绩确定、估值低位、政策支持的硬科技（半导体、低位AI算力）与新质生产力主线（电力基建、高端制造） 。
