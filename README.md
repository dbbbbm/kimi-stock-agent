## 使用的模型/AGENT
Claude code + kimi k2.5

## 手动操作
```
sh run_once.sh
```
然后和claude对话

## 命令
内置了5个命令
- `\analysis` 只做出分析，不操作
- `\operate`  分析，并直接做出操作
- `\update`   更新收益、持仓
- `\daily_archieve` 每日收盘总结
- `\weekly_archieve` 每周收盘总结

## 自动操作（3-21更新）
```
python scheduler.py
```
![1](./imgs/1.PNG)
支持的功能：
- 定时更新股票状态
- 定时运行`\operate`命令
- 持仓股票波动时触发`\operate`
- 每日收盘自动归档
- 每周收盘自动归档
